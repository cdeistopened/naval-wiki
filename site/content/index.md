# Wealth, happiness, leverage.

The structured knowledge base of Naval Ravikant - angel investor, philosopher-entrepreneur, AngelList founder. 160 episodes, 18 skills, 31 frameworks, ~327K words.

## What's inside

- **Frameworks** - Specific knowledge, leverage stacking, wealth creation, the happiness equation
- **Skills** - Decision skills you can run in Claude Code to apply Naval's thinking to your situation
- **Episodes** - Searchable archive of every episode

## Install the plugin

```bash
git clone https://github.com/cdeistopened/naval-wiki
cd naval-wiki
claude
```

Then type `/ask-naval` to get started.
